RC 低通滤波器实验总体介绍

本页面用于展示本次 RC 低通滤波器实验的背景、整体流程与网站功能结构。

01 实验背景

RC 低通滤波器是由电阻（Resistor，R） 和电容（Capacitor，C） 组成的被动滤波电路， 核心功能是：允许低频信号通过，衰减或阻断高频信号（包括高频噪声、干扰）， 是电子电路中最基础且应用广泛的滤波电路。

本实验旨在通过可视化方式展示 RC 电路的充电与放电过程， 理解时间常数 τ = RC 对波形变化的决定作用， 并通过前端动态曲线帮助学习者直观认识低通滤波器的工作原理与动态行为。

02 核心思路与流程

本实验通过构建理论模型、生成动态响应曲线并实现网页交互展示， 让用户能够直观观察 RC 电路在不同参数下的响应速度与滤波效果。 整体流程如下：

  1. 建立 RC 低通滤波模型
    • 采用标准的串联电阻 R 与电容 C 结构，即 V_in → R → V_out → C → GND
    • 对正弦波输入信号进行响应模拟
    • 依据公式构建输出电压的正弦波模型
  2. 生成不同参数条件下的理论动态波形（不含暂态过程）
    • 改变 R 或 C，观察相位和振幅变化引起的输出波形差异
    • 展示滤波器对高频信号的“振幅衰减”作用
    • 以坐标曲线形式可视化不同参数对应的V_out随时间变化曲线
  3. 构建网页交互系统
    • 通过 JavaScript 与 Chart.js 动态绘制响应曲线
    • 用HTML与CSS实现用户界面和交互
  4. 最终以可视化与交互方式呈现 RC 电路的核心特性，辅助课程理解
