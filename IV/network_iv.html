网站搭建相关介绍

本页面用于展示本次实验的网站搭建思路，结构和前端处理

01 基本建设

  • 网站前端使用 HTML, CSS和 JavaScript 开发，通过 Live Server 插件在 VSCode 中进行本地预览和调试。
  • HTML 部分构建网站基本框架，CSS 负责各式样式设计，最终 JavaScript 实现具体的交互，跳转功能。
  • 在“ I-V 曲线分析”子页面，我们引入了外部 streamlit 链接模拟进行动态交互。

02 核心思路与流程

本网站采用纯前端开发的方式，所有内容均通过 HTML、CSS 和 JavaScript 实现，无需后端支持。

  • 网站的所有文件（HTML、CSS、JavaScript、图片等）均存储在 GitHub 仓库中，通过 GitHub Pages 进行静态托管。
  • 用户访问网站时，浏览器直接从 GitHub Pages 加载静态文件，无需服务器动态处理。
  • IV 和 RC 两个模块分别展示了不同的实验内容，IV 模块展现 TCAD 仿真流程，RC 模块展示了 RC 电路的分析与仿真。
  • 通过模块化的文件结构（如独立的 CSS 和 JavaScript 文件），实现了代码的复用性和可维护性。
  • 整个网站的导航清晰，用户可以通过面包屑导航和功能卡片快速跳转到不同模块。
